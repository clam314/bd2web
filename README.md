# 棕色尘埃2 有缘之客 · 动态互动展示

一个用来玩《棕色尘埃2》有缘之客互动插画的网页，同时也能看全角色的 Spine 立绘动画。

打开页面不是看录屏，而是把游戏的 Spine 资源直接放进浏览器实时渲染：立绘会动，热区能点，阶段会切换，语音和音效跟着当前状态走。

[打开网页](https://clam314.github.io/bd2web/)

## 有缘之客

主页面就是「有缘之客」——尽量还原游戏里的互动方式：阶段切换、画面热区、点击/拖拽/长按/晃动触发对应动作，部分语音和音效会跟着当前角色状态走。

打开热区显示可以看到哪些地方能点；关掉热区后，就是一张沉浸式插画。默认打开最新收录的角色，也可以从左侧菜单搜索、切换到任意一位。

## 角色动画

[打开角色动画页面](https://clam314.github.io/bd2web/characters.html)

角色图鉴式的动画展示，覆盖上游素材仓库收录的全部角色。

每次打开都会随机抽一位角色和一套服装，页面会自动进入适合欣赏的展示状态。也可以从菜单里手动切换角色、服装、待机、互动和技能动作。

技能动画不做硬凑的完整演出复刻，只把资源里真正能播放的动作列出来，保持图鉴页该有的样子：干净、直接。

## 说明

这是一个非官方项目，主要用于学习和展示 Spine 动画播放效果。

页面播放的是 Spine 动画资源，不是普通视频。部分技能动画不会复刻游戏里的完整相机、粒子和后处理演出；部分互动因为浏览器拿不到游戏私有逻辑（缩圈判定、随机权重等），做了合理近似。

请勿公开传播游戏素材。
